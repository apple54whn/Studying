# Shiro

## 1 架构及简介

### 1.1 简介

Apache Shiro 是 Java 的一个安全（权限）框架。易用且可以用于JavaSE环境和JavaEE环境。它可以实现认证、授权、会话管理（SE也可以）、加密、与Web集成、缓存等。如下：
